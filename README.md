# Writing Utilities

A collection of Python scripts designed to assist in long-form writing and AI training workflows.

## 🚀 Overview
This repository contains tools developed to bridge the gap between creative writing and technical data validation. My goal is to use automation to ensure narrative consistency and linguistic precision.

## 📄 Current Tools
* **Text Analyzer (`text_analyzer.py`):** Scans documents for 'red-flag' words and provides basic frequency metrics. Useful for identifying overused words and improving prose clarity.

## 🛠️ Tech Stack
* **Language:** Python 3
* **Environment:** WSL2 (Ubuntu) / VS Code
* **Version Control:** Git / GitHub
